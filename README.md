

# 🚖 Namayatri Data Analysis & Dashboard Project

## 📚 Overview  
This project presents a **comprehensive data analysis and visualization dashboard** for the **Namayatri** ride-hailing service in **Bangalore**, India. It focuses on deriving actionable insights for **improving driver earnings, ride efficiency, and customer engagement**.

We used:
- **SQL Server Management Studio (SSMS)** for data cleaning and querying.
- **Power BI** for visual storytelling through dashboards.
- **Excel** for initial raw data processing.

The result? A **business intelligence dashboard** designed to help the Namayatri team make data-driven decisions.

---

## 🎯 Project Objectives  
- Analyze **ride patterns**, **customer behavior**, and **conversion rates**.
- Understand **revenue distribution**, **peak hours**, and **area-wise performance**.
- Provide **actionable insights** to improve **service efficiency** and **driver earnings**.
- Visualize complex data in an **interactive Power BI dashboard**.

## 📊 Dashboard Features & KPIs
<img width="574" alt="4" src="https://github.com/user-attachments/assets/be3a2332-4edd-4e09-a5fb-469af714b4f2" />

### 🔹 **Key Metrics (KPI Cards)**
- **Completed Rides**
- **Searches**
- **Estimates**
- **Quotes**
- **Driver Earnings**

### 🔹 **Conversion Rate Gauge**
- **Search to Quote**
- **Quote to Ride**
- **Ride Completion Rate**
  
A **gauge meter** visually tracks conversion percentages across different stages in the customer journey.

### 🔹 **Line Charts**
- **Trips vs Duration**: Helps understand ride time patterns.
- **Distance vs Duration**: Tracks efficiency and potential congestion issues.
- **Trips vs Fare**: Analyzes ride volumes relative to earnings.
<img width="551" alt="5" src="https://github.com/user-attachments/assets/5ae251d0-c6ee-4533-9b31-2e3057f2bc7f" />

### 🔹 **Assembly (Area) Wise Breakdown Table**
- Tabular representation of key metrics (rides, earnings, conversions) for different **Bangalore** areas.
- Helps identify high-performing and low-performing zones.

### 🔹 **Bangalore Area Map**
- Interactive **Power BI Map** showing:
  - **Number of trips per area**
  - **Demand hotspots**
  - **Performance of drivers by location**
